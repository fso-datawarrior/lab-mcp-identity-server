# Security

This repository is a **demonstration build**, not production software.

The real-mode path performs **destructive identity operations** (including revoke and deactivate) against whatever Okta organization you configure. Run it only against a throwaway developer org that has the demo fixture. Do not point it at a production directory.

To report a security concern or other issue, open a [GitHub Issue](https://github.com/fso-datawarrior/lab-mcp-identity-server/issues) on this repository.
